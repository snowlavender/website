#HTML 標籤

## 前言
html 檔是由一塊 html 標籤組成的，而每一個 html 標籤，通常都是雙標籤。用法就像這樣：

```html
<標籤名> 內容 </標籤名>
```
而一個最基本的 html 檔，一定會有 html 標籤

---

## html : 網頁檔標籤
> 說明: 一個網頁檔只會有一個這個標籤，它代表了整個網頁，使用方法如下：

> ```html
> <html>
	內容 ...
> </html>
> ```

---

## body : 網頁內容標籤
> 說明: 網頁上看得到的範圍，都屬於 body 的管轄範圍，而 body 通常在 html 裡面，用法：

> ```html
> <html>
	<body>
	blablabla...
	</body>
> </html>
> ```

---

## head : 網頁資訊標籤
> 說明: 網頁非看得到的範圍，都屬於 head 的管轄範圍，而 head 通常在 html 裡面，我們常見到的網頁title，就是放在 head  裡面，用法：

> ```html
> <html>
	<head>
		<title>
			I'm title
		</title>
	</head>
	<body>
	blablabla...
	</body>
> </html>
> ```

### 接下來我們會先專注於 `body` 內的標籤介紹，`head` 內的標籤之後再說。

---

## h1 : 抬頭文字標籤
> 說明: 相當於我們在 word 裡使用的 heading1，它會讓字變大變粗，就像一個文章的抬頭一樣，用法：

> ```html
> <h1> as heading 1 </h1>
> ```
> 值得一提的是，除了 h1, 還有 h2, h3 ... h6 ，而通常 h4 開始就比一般文字還要小，所以通常我們不會用。

---

## p : 段落標籤
> 說明: 包起來的文字代表一個文章的段落，用法：

> ```html
> <p> as paragraph </p>
> ```

---
## strong : 粗體標籤
> 說明: 這我想就不用多說了，就是把字變粗，用法：

> ```html
> <strong> as strong </strong>
> ```

---
## i : 斜體標籤
> 說明: 就是把字變斜，用法：

> ```html
> <i> as Italics </i>
> ```

---
## u: 底線標籤
> 說明: 就是把字加底線，用法：

> ```html
> <u> as underline </u>
> ```

---

## big: 底線標籤
> 說明: 就是把字變大，用法：

> ```html
> <big> as big </big>
> ```

---

## small: 底線標籤
> 說明: 就是把字變小，用法：

> ```html
> <small> as small </small>
> ```

---

## br: 斷行標籤
> 說明: 把一個內容斷行（注意，不同段落看起來也很像斷行，但意義不同，一個段落裡可以有很多個斷行，但它們還是屬於「同一個段落」

> ```html
> AAAAA <br /> BBBBB
> ```
注意，它是html標籤裡少見的「單標籤」，你可以寫成 <br /> 或是 <br>。
值得一值的是，在原始碼裡面，不管你加多少空白或是斷行，在瀏覽器顯示時，它都會示為「1個空白」所以真的要使用斷行，請用 br

---

## 標籤使用注意事項：
標籤可以包著使用，可以讓內容有多種變化，例如

>```html
<strong><i> 我是粗斜體 </i></strong>
>```

當然，你想的話，你可以加很多種

>```html
<strong><i><u><big> 我是粗斜底線又變大的字 </big></u></i></strong>
>```

值得注意的是，粗體裡面再包粗體，`字不會變更粗`，斜體裡面包斜體`字不會變更斜`…但 `big裡面包big，字會越包越大，small也是越包越小`



---

## span: 文字標籤
> 說明: 它沒有預設樣式，單純地把文字包起來而已。

> ```html
> <span> as span 1</span> <span> as span 2</span>
> ```

---

## div: 區塊標籤
> 說明: 它也沒有預設樣式，單純地把文字包起來而已。但不同的是，一個 div 會佔掉「一個區塊」，也就是說，不同的 div，看起像就像不同段落一樣。

> ```html
> <div> as div 1</div> <div> as div 2</div>
> ```
> 請跟 span 比較看看

---

## 註解標籤
> 說明:  註解標籤用在原始碼中作為文字筆記的區塊，被註解文字不會有任何效用（瀏覽器不會理它），單純給開發者看的。

> ```html
> <!-- 這邊可以打任何文字，瀏覽器不會理它 -->
> <!-- 
>     你也可以打多行文字
>     這是第二行
>     這是第三行
>  -->
> ```
> ps:  在 sublime 按 [cmd] + / 可以把該行註解


---

##  ul: 項目清單塊 = Unordered List
> 說明:  用來呈現項目類的清單，但裡面要包 `li` 標籤才可以呈現項目。

> ```html
> <ul>
>   <li> 項目一 </li>
>   <li> 項目二 </li>
>   <li> 項目三 </li>
> </ul>
> ```
> 項目裡面可以再包項目，而項目的符號也會跟著變
> ```html
> <ul>
>   <li> 項目一 </li>
>   <li> 項目二 </li>
>   <li> 項目三 </li>
>   <li>
>     <ul>
>       <li>子項目一</li>
>       <li>子項目二</li>
>     </ul>
>   </li>
> </ul>
> ```

---

##  ol: 序號清單塊 = Ordered List
> 說明:  用來呈現序號類的清單，但裡面要包 `li` 標籤才可以呈現項目。

> ```html
> <ol>
>   <li> 數字項目一 </li>
>   <li> 數字項目二 </li>
>   <li> 數字項目三 </li>
> </ol>
> ```


---

##  li: 清單項目 = list item
> 說明:  用來當作 `ul` 或是 `ol` 的項目使用

> ```html
> <ul>
>   <li> 數字項目一 </li>
>   <li> 數字項目二 </li>
> </ul>
> 
> <ol>
>   <li> 數字項目一 </li>
>   <li> 數字項目二 </li>
> </ol>
> ```

---

##  table: 表格標籤
> 說明:  代表一個表格，裡面會有多個列 (ROW)

> ```html
> <table>
>   <tr> ... </tr>
>   <tr> ... </tr>
> </table>
> ```

---

##  tr: 表格列標籤
> 說明:  代表一個表格列，裡面會有多個欄位 (COLUMN)

> ```html
> <table>
>   <tr>
>     <td>第1列 欄位1</td>
>     <td>第1列 欄位2</td>
>   </tr>
>   <tr>
>     <td>第2列 欄位1</td>
>     <td>第2列 欄位2</td>
>   </tr>
> </table>
> ```
> ps: 每一列的欄位數要一樣，不然會有不對稱的情況出現，比方說：
> 第一列3欄，第二列就一定要3欄。

---

##  td: 表格欄
> 說明:  代表一個表格欄，裡面可以有任何內容。

> ```html
> <table>
>   <tr>
>     <td>第1列 欄位1</td>
>     <td>
>       <ul>
>         <li> 項目一 </li>
>         <li> 項目二 </li>
>       </ul>
>     </td>
>   </tr>
> </table>
> ```
> ps: 每一列的欄位數要一樣，不然會有不對稱的情況出現，比方說：
> 第一列3欄，第二列就一定要3欄。

---

##  a: 超連結 = anchor
> 說明:  這是網頁標籤裡，最偉大的標籤之一，它可以幫我們從一個網頁連到另一個網頁。

> ```html
> <a href="2.html">連到 2.html</a>
> ```
> href  = Hypertext(超文字) + REFerence(資源) 的縮寫。既然都叫資源，那資源端一定要存在。如果被連的那一端沒有網頁，那就會出現`找不到網頁` (也就是我們俗稱的404)
>
> ```html
> <a href="https://www.google.com.tw/">連到 Google</a>
> ```
> 當然你也可以連到任一個網頁，記得加上 http or https
> 
> ```html
> <a href="https://www.facebook.com/" target="_blank">另開 Facebbok</a>
> ```
> 幫 `a` 加上 `target` 屬性，讓我們可以把網頁開到指定的目標，如果 `target` 內容是 `_blank` 的話，它就會另開新頁，不會整頁導過去。

---

##  img: 圖片標籤
> 說明:  呈現一張有效的圖片

> ```html
> <img src="abc.jpg" />
> ```
> src = source， 跟 `a` 一樣，你可以指定網路上的任何資源。
> ```html
> <img src="http://thecatapi.com/api/images/get?format=src&type=gif" />
> ```
> 另外你也可以指定它的寬、高：
> ```html
> <img src="abc.jpg" width="100" />
> ```
> 注意的是，若我們只有指定 width 指定為 100，那高會隨著寬的改變而等比例縮小，而如果寬、高都指定的話，要注意可能會變型。


##  link: 戴入樣式標籤
說明:  注意！它是一個 `head` 內的標籤，它可以幫我們戴入 css 檔案。

> ```html
> <link rel="stylesheet" href="css/website.scss" />
> ```
> `rel = relationship`，是關聯類別的意思，`stylesheet`指的是我們連結的檔案是樣式檔 (Cascading Style Sheet) 也就是 CSS檔，`href` 就不用說了吧，就是資源檔的來源，跟 `a` 一樣可以指定專案內的檔案或是網路上的資源。
